# 设备管理页面

## 功能概述

设备管理页面提供了完整的NFC设备管理功能，包括设备的增删改查、配置管理、状态监控和批量操作。

## 核心功能

### 1. 设备列表展示
- 表格形式展示所有设备信息
- 显示设备编码、名称、类型、位置、状态等关键信息
- 实时显示设备在线状态和电池电量
- 支持分页浏览，默认每页20条

### 2. 设备搜索与筛选
- **关键词搜索**: 支持按设备名称或编码搜索
- **状态筛选**: 可按在线、离线、维护状态筛选
- **类型筛选**: 可按桌台、墙面、柜台、入口等类型筛选

### 3. 设备添加
点击"添加设备"按钮，填写以下信息：
- **设备编码**: 唯一标识符，创建后不可修改
- **设备名称**: 便于识别的设备名称
- **设备类型**: 选择设备安装类型
- **设备位置**: 设备的物理位置描述
- **设备描述**: 可选的补充说明

### 4. 设备编辑
- 点击"编辑"按钮可修改设备信息
- 设备编码不可修改
- 其他字段均可更新

### 5. 设备配置
点击"配置"按钮进入设备配置界面，可设置：

#### 触发模式
- **视频生成**: NFC触发后生成个性化视频
  - 需选择内容模板
- **优惠券**: 跳转到优惠券页面
  - 需配置跳转链接
- **WiFi连接**: 自动连接WiFi
  - 需配置WiFi名称和密码
- **好友添加**: 添加企业微信
  - 需配置企业微信二维码地址
- **菜单展示**: 展示电子菜单
  - 需配置跳转链接

#### 设备状态
- 在线: 设备正常工作
- 离线: 设备暂停服务
- 维护: 设备维护中

### 6. 设备删除
- 点击"删除"按钮删除设备
- 需要二次确认
- 删除操作不可恢复

### 7. 批量操作
- 勾选多个设备后可进行批量操作
- **批量启用**: 将选中设备状态设为在线
- **批量禁用**: 将选中设备状态设为离线

### 8. 实时刷新
- 点击"刷新"按钮手动刷新设备列表
- 页面每30秒自动刷新一次
- 确保设备状态信息及时更新

## 字段说明

### 设备类型
- **TABLE**: 桌台 - 餐桌上的设备
- **WALL**: 墙面 - 墙上安装的设备
- **COUNTER**: 柜台 - 收银台等柜台设备
- **ENTRANCE**: 入口 - 门口或入口处的设备

### 设备状态
- **在线(1)**: 设备正常工作，可接受触发
- **离线(0)**: 设备暂停服务
- **维护(2)**: 设备正在维护中

### 触发模式
- **VIDEO**: 视频生成模式
- **COUPON**: 优惠券模式
- **WIFI**: WiFi连接模式
- **CONTACT**: 好友添加模式
- **MENU**: 菜单展示模式

### 电池电量显示
- 绿色(>50%): 电量充足
- 橙色(20%-50%): 电量偏低
- 红色(<20%): 电量不足，需要充电

## API接口

页面使用以下API接口：

```javascript
// 获取设备列表
nfcApi.getDevices(params)

// 获取设备详情
nfcApi.getDevice(id)

// 创建设备
nfcApi.createDevice(data)

// 更新设备
nfcApi.updateDevice(id, data)

// 删除设备
nfcApi.deleteDevice(id)

// 获取模板列表（用于配置）
contentApi.getTemplates(params)
```

## 使用建议

1. **定期检查**: 定期查看设备状态，确保设备在线
2. **电量监控**: 关注电池电量低的设备，及时充电
3. **合理配置**: 根据实际需求配置合适的触发模式
4. **位置标识**: 设置清晰的位置描述，便于定位设备
5. **批量管理**: 利用批量操作功能提高管理效率

## 技术栈

- Vue 3 Composition API
- Element Plus UI组件库
- Pinia状态管理
- Vue Router路由管理

## 注意事项

1. 设备编码创建后不可修改，请谨慎填写
2. 删除设备操作不可恢复，请确认后再操作
3. WiFi密码等敏感信息请妥善保管
4. 批量操作会影响多个设备，请谨慎使用
5. 设备配置修改后即时生效，请确保配置正确
