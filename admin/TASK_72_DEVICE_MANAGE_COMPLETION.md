## API接口使用

页面使用了以下API接口：

```javascript
// NFC设备API
nfcApi.getDevices(params)      // 获取设备列表
nfcApi.getDevice(id)           // 获取设备详情
nfcApi.createDevice(data)      // 创建设备
nfcApi.updateDevice(id, data)  // 更新设备
nfcApi.deleteDevice(id)        // 删除设备

// 内容模板API
contentApi.getTemplates(params) // 获取模板列表
```
