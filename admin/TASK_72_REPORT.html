核心目标

根据需求5.2，创建Vue管理后台的设备管理页面，提供设备列表、设备绑定、设备配置、状态监控等完整功能。实现了友好的用户界面和完善的交互体验。

✨ 功能实现

1. 设备列表管理

数据展示: 表格形式展示设备信息，支持排序和分页
状态显示: 彩色标签显示在线/离线/维护状态
电量监控: 进度条显示电池电量，颜色区分状态
实时更新: 30秒自动刷新，保持数据实时性

2. 搜索与筛选

关键词搜索: 支持按设备名称或编码搜索
状态筛选: 可按在线、离线、维护状态筛选
类型筛选: 可按桌台、墙面、柜台、入口筛选
组合筛选: 支持多条件组合筛选

3. 设备操作

添加设备: 表单验证，确保数据完整性
编辑设备: 数据回填，修改设备信息
删除设备: 二次确认，防止误删
批量操作: 批量启用/禁用设备

4. 设备配置

触发模式: 支持5种触发模式配置
动态表单: 根据模式显示对应配置项
模板选择: 视频模式支持选择内容模板
即时生效: 配置保存后立即生效

🚀 技术亮点

1. Vue 3 Composition API

// 使用 setup 语法，代码更简洁 <script setup> import { ref, reactive, onMounted } from 'vue' // 响应式数据 const deviceList = ref([]) const pagination = reactive({ page: 1, limit: 20, total: 0 }) // 生命周期钩子 onMounted(() => { loadDevices() }) </script>

2. 表单验证

// Element Plus 表单验证规则 const rules = { device_code: [ { required: true, message: '请输入设备编码' }, { min: 3, max: 50, message: '长度在 3 到 50 个字符' } ], device_name: [ { required: true, message: '请输入设备名称' } ] }

3. 动态表单

<!-- 根据触发模式动态显示配置项 --> <el-form-item label="内容模板" v-if="configForm.trigger_mode === 'VIDEO'"> <el-select v-model="configForm.template_id"> <el-option label="默认模板" :value="0" /> </el-select> </el-form-item>

4. 实时刷新

// 定时器实现自动刷新 let refreshTimer = null onMounted(() => { // 每30秒自动刷新 refreshTimer = setInterval(() => { loadDevices() }, 30000) }) onBeforeUnmount(() => { if (refreshTimer) { clearInterval(refreshTimer) } })
