3. 核心功能实现

3.1 请求工具 (request.js)

  • 自动添加JWT Token到请求头
  • 统一的错误处理和消息提示
  • 401自动跳转登录
  • 请求超时配置
  • Token管理函数（setToken, getToken, removeToken）

3.2 认证API (auth.js)

  • login() - 管理员登录
  • getUserInfo() - 获取用户信息
  • logout() - 退出登录
  • refreshToken() - 刷新Token

3.3 路由配置 (router/index.js)

  • 路由守卫：检查登录状态
  • 自动重定向未登录用户到登录页
  • 已登录用户访问登录页自动跳转
  • 保存原始路径用于登录后跳转

3.4 用户Store (store/modules/user.js)

  • 登录状态管理
  • 用户信息存储
  • Token持久化
  • 退出登录清理
  • 用户信息恢复

3.5 登录页面 (views/login/index.vue)

  • 用户名/密码表单
  • 表单验证
  • 记住我功能
  • 加载状态显示
  • 错误消息提示
  • 自动登录检查
  • 响应式设计
  • 渐变背景动画

4. API接口对接

POST /api/auth/login
请求体:
{
  "username": "admin",
  "password": "123456"
}

响应:
{
  "code": 200,
  "data": {
    "token": "jwt_token_string",
    "user": {
      "id": 123,
      "username": "admin",
      "nickname": "管理员",
      "avatar": "头像URL",
      "role": "admin"
    }
  }
}

5. 测试说明

5.1 启动开发服务器

cd D:\xiaomotui\admin
npm install
npm run dev

5.2 测试流程

  1. 访问 http://localhost:3000
  2. 自动跳转到 /login 登录页
  3. 输入用户名：13800138000（或 admin）
  4. 输入密码：123456
  5. 点击登录按钮
  6. 登录成功后跳转到 /dashboard
  7. 可以点击退出登录测试
