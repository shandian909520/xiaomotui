================================================================================
  核心功能
================================================================================

【认证流程】
1. 用户输入用户名和密码
2. 表单验证通过后提交
3. 调用 POST /api/auth/login 接口
4. 接收JWT Token和用户信息
5. 保存到localStorage和Pinia Store
6. 自动跳转到Dashboard页面

【路由守卫】
- 未登录用户访问受保护路由 → 跳转登录页
- 已登录用户访问登录页 → 跳转Dashboard
- Token过期 → 自动清除并跳转登录页

【状态管理】
- Token: localStorage + Pinia
- 用户信息: localStorage + Pinia
- 登录状态: Pinia getters

================================================================================
  API接口
================================================================================

【登录接口】
POST /api/auth/login

请求体:
{
  "username": "13800138000",  // 或 "admin"
  "password": "123456"
}

响应:
{
  "code": 200,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expires_in": 86400,
    "user": {
      "id": 123,
      "username": "admin",
      "nickname": "管理员",
      "avatar": "https://...",
      "role": "admin"
    }
  }
}
