# 商家推广功能完善 - 开发计划

## 一、项目概述

### 目标
完善商家端推广准备工作流程，包括：
1. 素材上传管理（图片、视频、音乐）
2. 视频合成与去重变体
3. 移动端商家助手
4. 推广配置向导
5. 数据统计

### 技术栈
- 后端：ThinkPHP 8
- 管理后台：Vue 3 + Element Plus
- 移动端：uni-app (Vue 3)
- 视频处理：FFmpeg

## 六、技术规范

### API 响应格式
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": 1234567890
}
```

### 文件命名规范
- 后端控制器：PascalCase，如 `PromoMaterial.php`
- 前端页面：kebab-case，如 `promo-material.vue`
- 数据库表：snake_case，如 `xmt_promo_materials`

### Git 提交规范
- feat: 新功能
- fix: 修复
- docs: 文档
- refactor: 重构
- test: 测试
