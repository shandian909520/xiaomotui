## 技术亮点

### 1. Promise Based API
所有API调用返回Promise，完美支持async/await语法，代码简洁优雅。

### 2. 类封装设计
使用ES6 Class封装请求逻辑，代码结构清晰，易于维护和扩展。

### 3. 条件编译
使用uni-app条件编译语法，完美处理不同平台的差异，一套代码多端运行。

### 4. 请求队列
Token刷新时自动暂存请求队列，刷新成功后自动重试，用户无感知。

### 5. 模块化设计
API按业务模块划分，职责清晰，便于维护和扩展。

### 6. 完善的日志
开发环境自动打印请求日志，方便调试。

## 使用示例

### 基础使用
```javascript
import api from '@/api'

export default {
  methods: {
    async handleLogin() {
      try {
        const res = await api.auth.wechatLogin()
        console.log('登录成功', res)
      } catch (e) {
        console.error('登录失败', e)
      }
    }
  }
}
```

### 自定义配置
```javascript
await api.content.getTaskStatus(taskId, {
  timeout: 60000,
  showLoading: false
})
```

### 自定义请求
```javascript
import { request } from '@/api'

const res = await request.post('/api/custom', data, {
  header: {
    'Custom-Header': 'value'
  }
})
```
