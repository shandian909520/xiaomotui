# 新手引导图片资源

本目录存放NFC触发新手引导的图片资源。

## 需要的图片

### 1. nfc-touch.png
**尺寸：** 600x400px
**内容：** 手机背面靠近NFC设备的示意图
**设计要点：**
- 展示手机背面（突出NFC芯片位置）
- 展示NFC设备（如餐桌贴纸）
- 用箭头或波纹线表示靠近距离（<5cm）
- 色调：清新、友好

### 2. auto-trigger.png
**尺寸：** 600x400px
**内容：** 手机震动并显示内容生成界面
**设计要点：**
- 手机屏幕显示"正在生成内容"
- 手机周围有震动波纹效果
- 界面显示进度条或加载动画
- 色调：活力、动感

### 3. scan-qr.png
**尺寸：** 600x400px
**内容：** 使用扫码功能扫描设备二维码
**设计要点：**
- 手机相机界面扫描二维码
- 突出显示"扫码"按钮位置
- 二维码示例（可以是模糊的）
- 色调：简洁、清晰

## 临时占位方案

在设计师提供正式图片前，可以使用以下方案：
1. 使用Emoji/Icon作为占位符（已在代码中实现）
2. 使用纯色背景+文字说明
3. 使用简单的线条插画

## 图片格式

- **格式：** PNG（支持透明背景）
- **压缩：** TinyPNG压缩后上传
- **命名：** 全小写，用短横线分隔
