# TabBar图标说明

本目录用于存放TabBar底部导航栏图标。

## 所需图标文件

请准备以下8个图标文件(建议尺寸: 81x81 像素):

### 1. 首页图标
- home.png - 首页未选中状态(灰色)
- home-active.png - 首页选中状态(紫色 #6366f1)

### 2. 素材图标
- material.png - 素材未选中状态(灰色)
- material-active.png - 素材选中状态(紫色 #6366f1)

### 3. 数据统计图标
- stats.png - 数据未选中状态(灰色)
- stats-active.png - 数据选中状态(紫色 #6366f1)

### 4. 个人中心图标
- user.png - 个人中心未选中状态(灰色)
- user-active.png - 个人中心选中状态(紫色 #6366f1)

## 设计规范
- 推荐尺寸: 81x81 像素
- 格式: PNG
- 背景: 透明
- 未选中: #7A7E83
- 选中: #6366f1
