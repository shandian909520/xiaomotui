```json

  "style": {
    "navigationBarTitleText": "内容预览",
    "navigationBarBackgroundColor": "#6366f1",
    "navigationBarTextStyle": "white"
  }
}
```

### 2. API集成

页面使用现有的API模块：

```javascript
import api from '../../api/index.js'

// 使用API
api.content.getTaskDetail(taskId)
api.content.saveToAlbum(url, type)
api.content.shareContent(data)

```

### 3. 页面跳转

```javascript
})
```
