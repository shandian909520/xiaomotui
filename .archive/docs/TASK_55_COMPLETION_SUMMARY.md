## 核心指标计算公式

### 1. 内容传播指数
```
传播指数 = (浏览量 × 1 + 分享量 × 3 + 点赞量 × 2 + 评论量 × 4) / 总数 / 4 × 100
范围: 0-100
```

### 2. 转化率
```
转化率 = (转化数 / 触发数) × 100%
```

### 3. ROI（投资回报率）
```
ROI = ((收益 - 成本) / 成本) × 100%
```

### 4. 用户留存率
```
留存率 = (第N天仍活跃用户数 / 第1天新增用户数) × 100%
```

### 5. 内容质量分数
```
质量分数 = (平均评分/5 × 100 × 40%) + (传播指数 × 30%) + (转化率 × 30%)
范围: 0-100
```
