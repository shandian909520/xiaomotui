## 使用说明

### 部署前准备

1. **复制配置文件**
   ```bash
   cp .env.production.example .env.production
   ```

2. **编辑配置文件**
   ```bash
   vim .env.production
   ```

3. **替换占位符**
   - 数据库配置：YOUR_PRODUCTION_DB_HOST等
   - Redis配置：YOUR_REDIS_HOST等
   - JWT密钥：使用`openssl rand -base64 64`生成
   - 云服务密钥：从各服务商获取
   - 域名配置：替换为实际域名

4. **设置文件权限**
   ```bash
   chmod 600 .env.production
   ```

5. **验证配置**
   - 测试数据库连接
   - 测试Redis连接
   - 测试云服务配置

### 安全注意事项

1. 所有占位符必须替换为实际值
2. 使用强密码和强密钥
3. 确保配置文件权限为600（仅所有者可读写）
4. 不要将.env.production提交到版本控制系统
5. 定期更新密钥和密码
