## 核心功能详解

### 1. 文件验证机制

**多层验证：**
- 文件存在性检查
- 文件格式验证（根据扩展名）
- 文件大小验证（各类型不同限制）
- 媒体文件特定验证（分辨率、时长、比特率）
- 内容安全检查（可选）

### 2. 素材导入流程

```
上传文件 → 验证文件 → 提取元数据 → 上传OSS → 生成缩略图 → 保存数据库 → 自动审核
```

### 3. 批量导入优化

- 支持多文件同时上传
- 分块处理大批量文件
- 可选队列异步处理
- 详细的导入结果统计

### 4. ZIP导入特性

- 自动解压ZIP文件
- 智能识别文件类型
- 按类型分组导入
- 自动清理临时文件

### 5. 审核机制

**自动审核：**
- 文件大小检查
- 敏感词过滤
- 基础规则验证

**人工审核：**
- 视频内容审核
- 版权合规检查
- 质量审核

### 6. 标签搜索

- 多标签匹配
- 匹配度排序
- 权重综合排序
- 结果缓存优化

## 技术亮点

### 1. 灵活的验证规则

每种素材类型都有独立的验证规则配置，可通过配置文件灵活调整。

### 2. 元数据提取

根据不同素材类型自动提取相关元数据：
- 视频：时长、分辨率、编码格式
- 音频：时长、比特率、编码格式
- 图片：宽度、高度、类型
- 文本：长度、行数、编码

### 3. OSS存储集成

- 支持阿里云OSS
- 自动生成存储路径
- 支持自定义域名
- 上传超时控制

### 4. 缓存优化

- 分类列表缓存
- 标签搜索结果缓存
- 可配置缓存时间
- 缓存键前缀隔离

### 5. 错误处理

- 统一的返回格式
- 详细的错误日志
- 友好的错误提示
- 异常捕获和恢复

## 文件大小限制

| 素材类型 | 支持格式 | 最大大小 | 特殊限制 |
|---------|---------|---------|---------|
| VIDEO | mp4, mov, avi, flv | 500MB | 时长1-60秒 |
| AUDIO | mp3, wav, aac, m4a | 50MB | 时长1-30秒 |
| IMAGE | jpg, jpeg, png, gif | 10MB | 最小800x600 |
| TEXT_TEMPLATE | txt, json | 1MB | 10-5000字符 |
| TRANSITION | mp4, mov, webm | 20MB | 最长3秒 |
| MUSIC | mp3, wav, aac, m4a, flac | 100MB | 10-600秒 |
