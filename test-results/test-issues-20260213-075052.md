### ISSUE-006: 系统管理相关接口中间件配置错误
- **问题ID:** ISSUE-006
- **关联功能点:** FP-015, FP-016, FP-017
- **问题描述:** 所有系统管理接口(/api/admin/*)都返回"class not exists: admin"错误
- **复现步骤:**
  1. 使用管理员账号登录
  2. 调用以下任一接口:
     - GET /api/admin/users
     - GET /api/admin/settings
     - GET /api/admin/operation-logs
  3. 观察返回结果
- **实际结果:** HTTP 500, 错误信息: "class not exists: admin"
- **预期结果:** 应该能够访问系统管理功能
- **API响应:**
  ```json
  {
    "code": 500,
    "msg": "class not exists: admin",
    "file": "D:\\xiaomotui\\api\\vendor\\topthink\\think-container\\src\\Container.php",
    "line": 383
  }
  ```
- **严重程度:** 致命
- **发现时间:** 2026-02-13 07:56:01-07:56:06
- **修复建议:**
  1. 检查路由配置文件(app.php)第391行的中间件配置: `->middleware([... , 'admin'])`
  2. 'admin'应该是一个中间件类名,而不是字符串'admin'
  3. 可能需要创建AdminAuth中间件,或使用现有的Auth中间件的admin角色检查功能
  4. 建议改为: `->middleware([\app\middleware\AdminAuth::class])` 或删除该中间件参数
